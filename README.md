# Travel Buddy

Travel Buddy is a web application for organizing trips through a simple and intuitive interface.

## Features
- Displays trip cards with destination information
- Responsive layout for different screen sizes
- Clean UI design using HTML and CSS
- (In progress) Adding interactivity with JavaScript

## Tech Stack
- HTML
- CSS
- JavaScript (in progress)

## Status
This project is currently in development. More features like trip management, itinerary tracking, and user interaction are being added.
